Create Subscription

Creates a new subscription for recurring payments. The subscription can be configured with various parameters including frequency, duration, and amount.

POST https://api.onepay.lk/v1/subscription

REQUEST PARAMETERS

Field Type Description
plan_id String The identifier of the subscription plan
customer_id String The unique identifier of the customer
start_date String The start date of the subscription (ISO 8601)

CREDENTIALS

REQUEST


{
    "plan_id": "plan_basic_monthly",
    "customer_id": "cust_12345",
    "start_date": "2024-03-15T00:00:00Z",
    "app_id": "80NR1189D04CD635D8ACD",
    "hash": "GR2P1189D04CD635D8AFD"
}

RESPONSE

200 400 403

Get Subscription

Retrieves the details of an existing subscription. You need to supply the unique subscription identifier that was returned when the subscription was created.

GET https://api.onepay.lk/v1/subscription/{id}

PATH PARAMETERS

Parameter Type Description
id String The unique identifier of the subscription

CREDENTIALS

REQUEST

GET /v1/subscription/{id}

RESPONSE

200 404
